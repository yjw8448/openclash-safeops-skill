# WorkBuddy Output Checklist

Every WorkBuddy repair answer should include:

1. What KB/playbook was used.
2. Current diagnosis.
3. Evidence from commands.
4. Risk level.
5. What was changed, if anything.
6. Backup directory.
7. Watchdog status.
8. Verification result.
9. Rollback command.
10. What user should click/test in LuCI or on client device.
